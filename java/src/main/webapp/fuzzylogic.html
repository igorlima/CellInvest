A modelagem fuzzy consiste i) na definição dos universos de discurso das variáveis linguísticas do sistema; ii) na definição do número de termos primários e graus de pertinência dos conjuntos difusos que representam cada termo; iii) na determinação das regras que formam o algoritmo de controle; e iv) a definição de parâmetros de projeto, como método de inferência, lógica a ser empregada, forma de defuzzificação e atuação do controlador.

A modelagem das Redes Neurais Artificiais é baseada na modelagem fuzzy, onde cada máquina de inferência fuzzy possui uma rna correspondente. Outras rna’s são utilizadas como redes auxiliares, possibilitando assim uma comparação entre a modelagem fuzzy e a modelagem rna.

Objetivo

O objetivo deste trabalho é analisar os resultados obtidos de duas versões de um mesmo software, uma versão implementada usando a técnica Lógica Fuzzy e outra versão usando a técnica Redes Neurais Artificiais (RNA), ambas de inteligência computacional. Esse software ajudará a predizer o movimento do mercado de capitais e obter informações importantes para tomada de decisão por parte do investidor.
